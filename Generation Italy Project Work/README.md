# Statistical correlation between market drivers and various stock characteristics
This is the final Project Work for the "Junior Data Engineer" DATITA031 course of Generation Italy  
The task was the following:<br/>
IT: "Il gruppo di ricerca del Dipartimento di Scienze Economiche vuole esaminare come variabili di mercato e caratteristiche delle azioni siano correlate tra loro.  
Si richiede di analizzare le relazioni tra variabili chiave per comprendere pattern di mercato e:  
- determinare se esistono correlazioni tra la volatilità delle azioni e il settore industriale;
- valutare se l’andamento delle azioni è stato influenzato da avvenimenti storici"
<br/>
EN: "The Department of Economic Sciences' research group wants to study how the market drivers and various stock characteristics present statistical correlation.  
You are tasked with the analysis of the relations between the key variables to describe market patterns and:  
- determine if the correlations between stock volatility and the market sector exist;
- find out if the trend of the stocks is influenced by historical events"
<br/>
What was done:  
- Data cleaning and analysis with python;
- upload datasets on MySQL server;
- server queried through RESTful API with Postman and HTML website;
- Power BI dashboard;
- Final dissertation on the project in italian
